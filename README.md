# Shortify

Cette application web vous permet de générer des extraits de partitions (aussi appelés "incipits") pour la musique traditionnelles québécoise et irlandaise.

L'application est disponible [ici](https://ludovical.github.io/Shortify/).

---

This web app renders small portions of sheet music (also known as 'incipits') for the Quebec and Irish trad music repertoire.

The app is available [here](https://ludovical.github.io/Shortify/).
